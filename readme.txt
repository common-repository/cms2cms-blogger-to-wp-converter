Blogger to WordPress automated data migration plugin. Convert Blogger to WordPress easily and fast.

== Description ==

You’ve got a blog on Blogger. After finding out that WordPress grants you more blogging opportunities you want to move to WordPress. On your Blogger blog there are dozens of posts and media attached and now you’re thinking how to quickly and safely move them to WordPress. Then this plugin is worth considering.

= HOW THIS PLUGIN HELPS: =
- Running Free Demo Migration to preview the results first;
- Average migration time lasts for 15 min. Note: timing depends on how many content items you’re up to move from Blogger to WordPress;
- Migrate the content you want. Map the data and choose how the migrated content lives and looks on WordPress;
- Migration price depends on the content amount you want to migrate to WordPress.


= WHAT WILL BE MOVED FROM BLOGGER TO WORDPRESS? =
* Pages and posts, content included there, content creation date and content author;
* Images included in the content body;
* Comments, comment author and comment creation date;
* Attachments (images and videos);
* Labels.


= Preserved SEO juice after the migration =
* Automated 301 redirects from the existing Blogger blog to the new WordPress website.

= COMPATIBLE WITH: WORDPRESS 4.1.X – 5.0.X. =
- Our support team is available live 24/7.
- For more customized migration type, CMS2CMS offers data migration service packages to order.

== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated Blogger To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide your Existing website URL address.
7. Click ‘Verify Connection’. You’ll be redirected to the CMS2CMS website to proceed with your migration.
